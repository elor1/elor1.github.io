Cadere

Cadere is a game I'm currently working on with a friend in Unreal Engine 4.

Although it is currently only in the early stages of development, we intend to create an open world survival game. The player will have to scavenge for resources, craft weapons and materials, and build hideouts to survive against enemies for a number of nights.

Some of the features I've worked on include:

  • Inventory/item drop system
  • Building system
  • AI behaviour trees
  • Basic punching/shooting combat
