Beat Defender

Beat Defender is a game I made in Unity for the Grads in Games Rising star competition 2021.

It is an endless wave-based shoot 'em up featuring gameplay that is influenced by the game's soundtrack using audio analysis and beat detection. My main goal for this project was to gain some more Unity experience as well as following a game through from idea to release.

One of the main challenges of this project was having to build upon an existing code base. We were given a basic framework which allowed the player to select between a series of options, which I decided to expand to an upgrade menu between rounds. This meant that after the player had defeated a wave of enemies they could decide to increase their stats to help them continue.

Other techniques used include:

  • Procedurally generated maps using cellular automata
  • A* enemy pathfinding
