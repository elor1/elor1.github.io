Beat Defender

Beat Defender is a game I made in Unity for the Grads in Games Rising star competition 2021.

It is a endless wave-based shoot 'em up game featuring gameplay that is influenced by the game's soundtrack using audio analysis and beat detection. This analysis is used to adjust enemy stats and spawn times so that the game's difficulty increases as the music gets more intense.

I used cellular automation to procedurally generate the map before using a flood fill algorithm to decide to keep certain wall or room regions.

The enemies use A* pathfinding to navigate the map and move towards the player

At the end of each wave, an upgrade screen is displayed allowing the player to improve their stats to help in taking on the next wave.
